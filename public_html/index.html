Home

Welcome to my personal homepage. This is where I display some of my work in Computer Science and Software Development as well as other interest projects. This website is a work in progress so please bear with me as I add content over the coming months. Many of the projects showcased on this site are also available as public repositories on my Personal GitHub so please feel free to browse these. I do ask however that you request permission for any use of my work as I do not provide public licences for these repositories. Limited licensing will be considered on a case by case basis.

In the sections below, you will find some of the projects which I have been recently working on, and some which I have yet to publish. Where projects corespond to public repositories on GitHub, links to both the repository and the project page should be available. Projects which are not GitHub repositories will provide only a project page link whilst projects which are not yet ready for publication may or may not furnish a project page link.
